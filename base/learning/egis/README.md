# 教育関係者向けGIS演習教材
本教材は、大学の教員養成系科目や教員研修の受講者を対象としたGISの教材です。事前にGISを講義と演習（QGISやArcGISを使用する演習）で1時間程度学習した人を対象にしています。GISの基礎知識・各ソフトウェア・アプリケーション基本操作は解説していません。
