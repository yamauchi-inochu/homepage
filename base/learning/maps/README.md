### 断面図
A-Bの横軸と0-10の縦軸が交わる点の標高値を読み取り、上から２マス目より下に点をプロットし、それをつないで、地形断面図を作成する。縮尺は地図と対応する。グラフを作成する際に、高さ倍率も考慮する。
